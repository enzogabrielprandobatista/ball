# Steel Ball Race — The Great Race

Jogo de corrida inspirado em JoJo's Bizarre Adventure Part 7, desenvolvido em HTML5 Canvas puro.

## Como jogar

### Controles (teclado)

| Tecla | Ação |
|-------|------|
| `←` `→` | Mover para esquerda / direita |
| `↑` ou `Z` | Pular |
| `↓` | Montar / Desmontar cavalo |
| `X` | Atacar |
| `C` | Especial |
| `P` | Pausar |

### Controles (celular)

Na tela touch aparece um gamepad virtual com D-Pad e botões de ação.

## Personagens

| Personagem | Ataque (X) | Especial (C) |
|------------|------------|--------------|
| **Johnny Joestar** | Dispara pregos (Tusk) | Pregos em leque / ricochete / rotação |
| **Gyro Zeppeli** | Lança Steel Ball teleguiada | 3 Steel Balls com homing |
| **Diego Brando** | Dispara balas / garras | Ativa Scary Monsters (forma dino) |

### Diego Brando — Scary Monsters
- Pressione `C` para transformar em dinossauro
- Na forma dino: velocidade e pulo aumentados, ataque corpo a corpo + balas teal
- **Não é possível montar cavalo na forma dino**
- A transformação dura mais tempo conforme o nível evolui
